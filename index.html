Minecraft Dungeons Save Editor (Web)

Browser-only editor for `.dat` and `.json` saves Port Version: 1.0.0 Dirty: No
Repository Issues Original Project All Editors Home

Drop a Minecraft Dungeons save file to edit locally in your browser. Nothing uploads to a server. This port supports encrypted `.dat` files and decrypted `.json` files, plus direct export back to game-compatible formats.

Deployment Notes + Default Save Locations

Save Files (.dat)

Open a character file from this default save root. Primary filename pattern is `*.dat`:

%HOMEPATH%\Saved Games\Mojang Studios\Dungeons\<16-digit-id>\Characters\*.dat Expected primary cloud file name: character `*.dat` in `Characters`.

Game Content (Paks)

Useful if you compare item/content data outside the editor:

Launcher: %LOCALAPPDATA%\Mojang\products\dungeons\dungeons\Dungeons\Content\Paks Steam: %PROGRAMFILES(X86)%\Steam\steamapps\common\MinecraftDungeons\Dungeons\Content\Paks Xbox app: C:\XboxGames\Minecraft Dungeons\Content\Dungeons\Content\Paks

Cloud Save Notes

Watch for three variants before editing:

1) Active local synced save 2) Cloud conflict/duplicate save copy 3) Local backup copy (.bak or renamed .dat)
